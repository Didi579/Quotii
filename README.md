# Quotii

Import quote calculator for costing, logistics, pallets, duties, and margin pricing.

## Files

- `index.html` - main Quotii calculator page
- `manifest.json` - PWA app metadata
- `service-worker.js` - offline cache support
- `icons/quotii.png` - app icon

## Publish With GitHub Pages

1. Create a new GitHub repository, for example `quotii`.
2. Upload everything inside this folder to the repository root.
3. In GitHub, open `Settings` > `Pages`.
4. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. GitHub will give you a website URL after deployment.

## Open On iPhone

1. Open the GitHub Pages URL with Safari.
2. Tap the Share button.
3. Tap `Add to Home Screen`.
4. Quotii will appear on the iPhone home screen with the app icon.

## Note

For full PWA behavior, use the HTTPS GitHub Pages URL. Opening `index.html` directly from local files will not enable all PWA features.
